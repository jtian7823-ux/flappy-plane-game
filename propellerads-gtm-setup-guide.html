PropellerAds GTM 設定指南

含 localStorage fallback 跨訪次歸因架構

id1 = aid id2 = tid utm_prid = visitor_id

🔗 投放連結格式

在 PropellerAds 後台的 Landing Page URL 欄位填入：

https://jtian7823-ux.github.io/flappy-plane-game/?utm_prid={SUBID}&id1=3899879&id2=154469

參數說明

參數對應值說明
utm_prid{SUBID}PropellerAds 自動替換的訪客 ID macro
id13899879廣告帳號識別碼（aid）
id2154469追蹤模板識別碼（tid）
⚠️ 三個參數缺一不可，缺少任何一個都無法正確回傳轉換

💾 localStorage 歸因邏輯

訪客進入頁面時，JS 自動執行以下優先順序讀取：

  • 1URL 參數（最優先）　讀取 utm_prid、id1、id2，同時寫入 localStorage 更新快取
  • 2localStorage（fallback）　訪客第二次入站、URL 無參數時，從快取讀取
  • 3都沒有 → 不觸發　無法歸因時，Deposit tag 不發送 conversion request

落地頁初始化代碼（加入 index.html）

(function() { var p = new URLSearchParams(location.search); var prid = p.get('utm_prid'); var id1 = p.get('id1'); var id2 = p.get('id2'); // 優先用 URL 參數，有值就寫入 localStorage if (prid) localStorage.setItem('_prid', prid); if (id1) localStorage.setItem('_id1', id1); if (id2) localStorage.setItem('_id2', id2); })();
✅ Cookie 有效期 30 天，localStorage 永久保留（除非用戶清除）

⚙️ GTM Tags 設定

TAG 1 PropellerAds Cookie — All Pages

讀取 URL 參數存入 Cookie，供舊版歸因備用。

<script> var p = new URLSearchParams(location.search); var prid = p.get('utm_prid'); var id1 = p.get('id1'); var id2 = p.get('id2'); // Cookie（30天） if (prid) { var exp = new Date(); exp.setTime(exp.getTime() + 30*24*60*60*1000); document.cookie = 'prop_visitor_id=' + prid + '; expires=' + exp.toUTCString() + '; path=/'; } // localStorage（同步更新） if (prid) localStorage.setItem('_prid', prid); if (id1) localStorage.setItem('_id1', id1); if (id2) localStorage.setItem('_id2', id2); </script>

TAG 2 PropellerAds Registration — Registration Trigger

<script> var visitorId = localStorage.getItem('_prid') || ('; ' + document.cookie).split('; prop_visitor_id=').pop().split(';')[0]; var aid = localStorage.getItem('_id1'); var tid = localStorage.getItem('_id2'); if (visitorId && aid && tid) { var xhr = new XMLHttpRequest(); xhr.open('GET', 'https://ad.propellerads.com/conversion.php' + '?aid=' + encodeURIComponent(aid) + '&pid=&tid=' + encodeURIComponent(tid) + '&visitor_id=' + encodeURIComponent(visitorId), true ); xhr.send(); } </script>

TAG 3 PropellerAds Deposit — Deposit Trigger

<script> var visitorId = localStorage.getItem('_prid') || ('; ' + document.cookie).split('; prop_visitor_id=').pop().split(';')[0]; var aid = localStorage.getItem('_id1'); var tid = localStorage.getItem('_id2'); if (visitorId && aid && tid) { var xhr = new XMLHttpRequest(); xhr.open('GET', 'https://ad.propellerads.com/conversion.php' + '?aid=' + encodeURIComponent(aid) + '&pid=&tid=' + encodeURIComponent(tid) + '&goal=2' + '&visitor_id=' + encodeURIComponent(visitorId), true ); xhr.send(); } </script>

🔄 更換 aid / tid

未來更換廣告帳號時，只需要修改投放連結的 id1 和 id2 值，GTM 代碼不需要改動。

1
修改投放連結 把 id1=NEW_AID&id2=NEW_TID 填入 PropellerAds 後台
2
GTM 自動接收新值 落地頁 JS 讀取新 id1/id2 → 存入 localStorage → GTM tag 使用新值
3
舊訪客自動更新 訪客下次帶新參數入站時，localStorage 會被覆蓋為新值
